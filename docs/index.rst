New York City's Charter, Laws and Code
=================================

We created accessible, searchable, mobile-friendly websites for New York City's charter, codes and rules because New Yorker's deserve better than what the city offers. It was last updated: February 4th, 2020. More information about this project at `WeGov.nyc/laws <http://research.wegov.nyc/laws>`_.
